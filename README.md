# umkm-backend
api untuk umkm restoran

## Instalasi
```
$ npm install
```
## Penggunaan


1. import data base, file database berada di app/database
    ```
    $ app/database
    ```

2. ubah konfigurasi data base (user name dan password) di 
    ```
    $ app/configs/database
    ```
3. import postman collection (opsional)
    ```
    $ /post-man
    ```
